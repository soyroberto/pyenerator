# Pyenerator: Python Password Generator

![Python Password generator (PPG)](https://github.com/soyroberto/pyenerator/blob/main/password_generation_process.png)

Pyenerator is a versatile command-line tool for generating strong, memorable passwords and passphrases. It offers two distinct modes: a traditional random character password generator and an XKCD-style passphrase generator. This tool is designed to be flexible, allowing users to customize the generated passwords to meet various security requirements.

## Features

Pyenerator includes a comprehensive set of features for both password generation modes:

### Traditional Mode

*   **Customizable Character Sets:** You can choose to include or exclude specific character types to meet password policy requirements.
    *   Numbers (0-9)
    *   Basic Symbols (!@#$%^&*)
    *   Special Characters ( ()[]{}|\\:;\"'<>,.?/~`)
*   **Variable Length:** Specify the desired length of the password.

### XKCD-Style Passphrase Mode

*   **Word-based Passphrases:** Generates passphrases using a list of random words, making them easier to remember.
*   **Large Wordlist:** Utilizes a Spanish wordlist with over 110,000 words for high entropy.
*   **Customizable Word Count:** Choose the number of words to include in your passphrase.
*   **Custom Separators:** Use any character to separate the words in the passphrase.
*   **Capitalization:** Option to capitalize the first letter of each word.
*   **Add Numbers and Symbols:** Append random numbers and symbols to the passphrase for additional security.

## Usage

### Installation

Clone the repository to your local machine:

```bash
git clone https://github.com/soyroberto/pyenerator.git
cd pyenerator
```

### Examples

#### XKCD-Style Passphrases

**Generate a basic 5-word passphrase:**

```bash
python3 main.py --xkcd
```

**Generate a passphrase with capitalization and numbers:**

```bash
python3 main.py --xkcd --capitalizar --agregar-numeros
```

**Generate a passphrase with a custom separator and word count:**

```bash
python3 main.py --xkcd --palabras 5 --separador "_"
```

**Generate a passphrase with all options enabled:**

```bash
python3 main.py --xkcd --palabras 6 --separador "." --capitalizar --agregar-numeros --agregar-simbolos
```

#### Traditional Random Character Passwords

**Generate a 12-character password with all character types:**

```bash
python3 main.py 12
```

**Generate a 12-character password excluding numbers:**

```bash
python3 main.py 12 --no-numeros
```

**Generate a 12-character password with only letters:**

```bash
python3 main.py 12 --no-numeros --no-simbolos --no-especiales
```

### Docker Usage

You can also use Pyenerator as a Docker container. First, build the Docker image:

```bash
docker build -t passwd .
```

Then, run the container with the desired options:

```bash
docker run passwd --xkcd --palabras 6 --separador "**"
docker run passwd 12 --no-especiales
docker run passwd 12 --no-simbolos --no-numeros
```

## Command Reference

| Argument | Short Form | Description |
| :--- | :--- | :--- |
| `--xkcd` | | Enable XKCD-style passphrase generation. |
| `longitud` | | Length of the password for traditional mode. |
| `--palabras` | | Number of words to use for XKCD passphrases (default: 5). |
| `--separador` | | Character to separate words in XKCD passphrases (default: "-"). |
| `--capitalizar` | | Capitalize the first letter of each word in XKCD passphrases. |
| `--agregar-numeros` | | Add random numbers to the end of XKCD passphrases. |
| `--agregar-simbolos` | | Add random symbols to the end of XKCD passphrases. |
| `--no-numeros` | `--non` | Exclude numbers from traditional passwords. |
| `--no-simbolos` | `--nos` | Exclude symbols from traditional passwords. |
| `--no-especiales` | `--noe` | Exclude special characters from traditional passwords. |

## Security Considerations

The security of a password or passphrase depends on its length and the size of the character or word set used. Here are some entropy estimates for passwords generated by Pyenerator:

### XKCD Passphrases

*   **4 words:** ~44 bits of entropy
*   **5 words:** ~55 bits of entropy
*   **6 words:** ~66 bits of entropy

### Traditional Passwords

*   **12 characters:** ~70 bits of entropy (using all character sets)
*   **16 characters:** ~94 bits of entropy (using all character sets)

## Word List

The XKCD-style passphrase generator uses a Spanish wordlist (`es.txt`) containing over 110,000 words. If this file is not found, it falls back to a basic built-in wordlist.
